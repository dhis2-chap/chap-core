# DAC2026

Workshop materials for DAC2026.

This series builds on the [Modeling Webinar Series](../kigali-workshop/kigali-webinar-series/index.md),
in particular sessions 2 and 3, where you set up a development environment and ran your
first model through CHAP.

## Guides

### [Make your model configurable](configurable-model.md)

Build a model whose behaviour can be configured from the outside: expose tunable options
and extra covariates, inspect the configuration spec with the CHAP CLI, and run an
evaluation with a specific configuration.
