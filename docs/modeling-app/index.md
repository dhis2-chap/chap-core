# Using Chap Core with DHIS2 Modeling App

Integrating the Modeling App with DHIS2 enables users to leverage advanced modeling and prediction tools directly within their health information system. This setup allows for seamless data exchange, improved decision-making, and the ability to run custom models on real-world health and climate data. By using Chap Core with DHIS2 via the Modeling App, organizations can enhance their analytics, automate workflows, and support evidence-based planning.
